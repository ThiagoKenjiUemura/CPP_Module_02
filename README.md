# CPP_MODULE_02
42 C++ - Module 02: Ad-hoc polymorphism, operator overloading, and the Orthodox Canonical Class Form. This module introduces fixed-point numbers, providing a practical way to understand precision, custom arithmetic operators, and the fundamental structure of robust C++ classes. Includes an optional BSP (Binary Space Partitioning) project.

Features
	Orthodox Canonical Form: Implementation of Default Constructor, Copy Constructor, Copy Assignment Operator, and Destructor.

	Fixed-Point Representation: Using 8 bits for the fractional part

	Operator Overloading:

	Comparison: >, <, >=, <=, ==, !=

	Arithmetic: +, -, *, /

	Increment/Decrement: Pre and Post ++ and --

	Utility Functions:

	toInt() and toFloat() for type conversion.

	min and max static member functions (including const versions).

Technical Details
	Language: C++ 98

	Compilation: make (Includes -Wall -Wextra -Werror flags).

	Key Concepts: Bit shifting, rounding (roundf), and reference-based performance optimization.
